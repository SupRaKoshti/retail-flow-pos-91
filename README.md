## About The Project

The **POS System** is a modern inventory and sales management platform built with Django REST Framework.
It allows businesses to efficiently manage:
- Products, categories, and stock
- Sales and invoices
- Customers and employees
- Reports and dashboards

This project is designed for small to mid-sized retail businesses that need a fast, web-based POS solution.
